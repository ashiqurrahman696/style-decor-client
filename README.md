# StyleDecor

### Project purpose

- StyleDecor is a modern appointment management system for a local decoration company that offers both in-studio consultations and on-site decoration services for homes and ceremonies.
- The app enables customers to explore decoration packages, check decorator availability, select date & time, choose a service mode (in-studio or on-site), make payments, and track the status of their booked services.
- It serves as a frontend client that integrates with backend APIs to manage bookings, payments, notifications, and staff schedules.

### Key features

- Browse decoration packages with images, descriptions, pricing, and service durations
- Real-time decorator availability and calendar view for booking
- Create bookings by selecting date, time, service mode (in-studio / on-site), and package options
- Search, filtering, and sorting of decoration packages
- User authentication / account management
- Secure payment flow (integration-ready for payment providers) and booking confirmation
- Booking management: view, reschedule, cancel, and track service status
- User accounts: booking history, favorites, saved addresses, and profile management
- Decorator/admin dashboard: manage availability, accept/decline bookings, and view daily schedules
- Responsive and accessible UI with RESTful API integration
